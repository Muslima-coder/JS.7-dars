Higher Order Function — bu boshqa funksiyalarni parametr sifatida qabul qiladigan yoki funksiya qaytaradigan funksiyadir.

Ularga: CallBack function, Compare function, Pridicate function
Map methodi

map() — massivning har bir elementiga callback function ni qo'llaydi va yangi massiv qaytaradi.

Callback function 3 ta argument oladi: item, index, array

Asl massiv o'zgarmaydi.

Har bir element ustida amal bajaradi va natijasini yangi massivga joylaydi.

ForEach methodi

forEach() — massivning har bir elementiga callback function ni qo'llaydi.

Callback function 3 ta argument oladi: item, index, array

Asosan konsolga chiqarish, DOM elementlarga o'zgartirish kabi ishlar uchun ishlatiladi.

Boshqa foydali metodlar

filter() — Berilgan shartni bajargan elementlarni yangi array qilib qaytaradi.

find() — Shartni birinchi bajargan elementni o‘zini qaytaradi.

findIndex() — Shartni birinchi bajargan elementning indexini qaytaradi.

sort() — Array elementlarini tartiblaydi (son, alfavit). Number uchun (a - b), matn uchun localeCompare ishlatiladi.

some() — Kamida 1 ta element shartni bajarsa true qaytaradi. (true/false)

every() — Barcha elementlar shartni bajarsa true qaytaradi. (true/false)
